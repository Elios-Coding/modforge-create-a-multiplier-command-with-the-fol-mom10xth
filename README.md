# CreateAMultiplierCommandWithTheFol

Adds per-player /multiplier controls for Paper/Spigot block drops without creating oversized stacks.

Generated by **ModForge AI** for Paper/Spigot-compatible Minecraft servers.

## Commands

- `/multiplier ON [value]` — enables your personal block drop multiplier and optionally sets the value.
- `/multiplier OFF` — disables your personal block drop multiplier without deleting the saved value.

## Build

```bash
./gradlew build
```

The compiled `.jar` will be placed in `build/libs/`.
